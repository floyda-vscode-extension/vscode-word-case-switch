# Word Case Switch  

> This is a VSCode extension that toggles the word between 'Snake', 'constant', 'camel' and 'pascal'  
> 这是一个VSCode扩展插件, 将单词在`snake`,`constant`,`camel`,`pascal`之间切换  

## Case Type  

```ts
enum CaseType {
    snake,
    constant,
    camel,
    pascal,
}
```

## keybindings

`alt+c`
